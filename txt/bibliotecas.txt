FLASK
Ele cria um site a onde todas as pessoas podem acessar, definindo uma aplicação única;

RENDER_TEMPLATE
Função utilizada para abrir arquivos em HTML usando Python;

REQUEST
Recebe as informações do usuário/admin. É nele que utilizamos o GET para receber, 
e o POST para enviar novas informações;

JSONIFY
Transforma os dados em Python para JSON (JavaScript Object Notation), 
uma maneira de mostrar as informações para o JavaScript poder ler e passar as informações;

JINJA2
Mecanismo de templates do Flask onde ele mistura Python com HTML para mostrar dados na página;

REDIRECT
Ele define para a onde o usuário deve ir quando clicar em outra página;

SESSION
Guarda as informações das tabelas de estoque e usuário enquanto ele estiver navegando no site;

URL_FOR
Ela torna mais prática a troca entre as páginas, e os links dos códigos;

MYSQL.CONNECTOR
É a biblioteca que conecta o Python com o MySQL

OS
Biblioteca do Python que mexe com arquivos e pastas, ele é utilizado na 
pasta UPLOADS tornando o processo mais rápido;

SECURE_FILENAME
É uma ferramenta de segurança para arquivos, trabalha em conjunto do OS 
para caso algum arquivo tiver nome estranho ele mude pra algo mais simples;
